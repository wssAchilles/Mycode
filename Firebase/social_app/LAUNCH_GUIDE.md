## 启动应用

### 方法 1: 命令行启动
在 `social_app` 目录下运行：
```bash
flutter run -d chrome
```

### 方法 2: VS Code 启动
1. 打开 `social_app/lib/main.dart` 文件
2. 按 F5 或点击 VS Code 右上角的运行按钮
3. 选择 Chrome 作为运行设备

## 🔧 故障排除

### 如果应用无法启动
1. 确保 Flutter 环境正确：
   ```bash
   flutter doctor
   ```

2. 清理并重新构建：
   ```bash
   flutter clean
   flutter pub get
   ```

### 如果 Data Connect 连接失败
1. 确认模拟器正在运行：
   - 检查终端显示 "Data Connect │ 127.0.0.1:9399"

2. 检查网络连接：
   - 访问 http://127.0.0.1:9399 查看模拟器状态
