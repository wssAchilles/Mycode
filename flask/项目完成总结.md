## 🚀 快速开始

### 方法一：运行演示版本（推荐）
```bash
cd d:\Code\flask
python demo_app.py
```
访问：http://localhost:5000

### 方法二：完整版本安装
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 初始化数据库
python run.py init-db

# 3. 创建管理员
python run.py create-admin

# 4. 启动应用
python start.py
```

### 方法三：Docker部署
```bash
# 单容器运行
docker build -t flask-app .
docker run -p 5000:5000 flask-app

# 或使用Docker Compose
docker-compose up -d
```
