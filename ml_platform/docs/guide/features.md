# 核心功能

## 🎨 算法与数据结构可视化

### 排序算法

支持以下排序算法的动态可视化:

| 算法 | 时间复杂度(平均) | 稳定性 | 特点 |
|------|---------------|--------|------|
| 冒泡排序 | O(n²) | 稳定 | 相邻元素比较交换 |
| 快速排序 | O(n log n) | 不稳定 | 分治思想 |
| 归并排序 | O(n log n) | 稳定 | 分治+合并 |
| 堆排序 | O(n log n) | 不稳定 | 堆数据结构 |

#### 使用方法

1. 进入"算法可视化"模块
2. 选择排序算法
3. 设置数据规模(10-200)
4. 点击"开始可视化"
5. 使用控制面板:
   - ⏯️ 播放/暂停
   - ⏭️ 单步执行
   - 🎚️ 调整速度

## 🖥️ 操作系统模拟器

### 进程调度算法

#### FCFS(先来先服务)
```text
特点: 简单,按到达顺序执行
优点: 公平,易实现
缺点: 等待时间可能很长(护航效应)
```

#### SJF(最短作业优先)
```text
特点: 选择执行时间最短的进程
优点: 平均等待时间最短
缺点: 可能导致饥饿,需预知执行时间
```

#### 时间片轮转(RR)
```text
特点: 每个进程分配固定时间片
优点: 响应时间好,公平
缺点: 上下文切换开销
```

### 死锁处理

#### 银行家算法
模拟资源分配安全性检查:
1. 定义进程和资源
2. 设置最大需求和已分配资源
3. 检查请求是否安全
4. 显示安全序列或死锁状态

## 🤖 机器学习实验平台

### 监督学习算法

#### 1. 线性回归
- **应用**: 房价预测、销量预测
- **参数**: 学习率、迭代次数、正则化系数
- **输出**: 拟合曲线、R²值、MSE

#### 2. 逻辑回归
- **应用**: 二分类问题(垃圾邮件检测)
- **参数**: 学习率、正则化类型
- **输出**: 决策边界、准确率、AUC

#### 3. 决策树
- **应用**: 分类和回归
- **参数**: 最大深度、最小样本数、分裂标准
- **输出**: 树结构可视化、特征重要性

#### 4. 随机森林
- **应用**: 集成学习
- **参数**: 树数量、最大特征数
- **输出**: 特征重要性排名

#### 5. 支持向量机(SVM)
- **应用**: 高维分类
- **参数**: 核函数、C参数、gamma
- **输出**: 决策边界、支持向量

#### 6. K近邻(KNN)
- **应用**: 分类和回归
- **参数**: K值、距离度量
- **输出**: 邻居关系图

### 无监督学习算法

#### K-Means聚类
1. 上传数据集
2. 设置聚类数K
3. 选择初始化方法
4. 观看迭代过程动画
5. 查看聚类结果

#### 降维算法
- PCA(主成分分析)
- t-SNE(t分布随机邻域嵌入)

### 实验流程

```text
数据上传 → 预处理 → 选择算法 → 参数调优 → 云端训练 → 结果分析
```

#### 数据要求
- 格式: CSV
- 大小: < 10MB
- 特征: 数值型(分类特征需编码)

#### 结果输出
- 📊 可视化图表
- 📈 性能指标
- 📝 详细报告
- 💾 模型下载

## 🎯 学习路径推荐

### 初学者路径
1. 从简单排序算法开始(冒泡、选择)
2. 理解基本数据结构(栈、队列)
3. 尝试简单的OS算法(FCFS)
4. 体验线性回归

### 进阶路径
1. 掌握高效排序算法(快排、归并)
2. 深入树和图算法
3. 理解复杂的调度算法
4. 实验多种ML算法并对比

### 考研路径
1. 系统学习408知识点
2. 通过可视化加深理解
3. 做题后用平台验证理解
4. 总结归纳形成知识体系
